# tru-mail
tru-mail - скрипт подключается по IMAP до почтового сервиса, читает последние письма и дает атрибуты и информацию о том кто его отправил.

Нужные модули:
    "dotenv": "^16.4.7",
    "express": "^4.21.2",
    "imap": "^0.8.19",
    "nodemailer": "^6.10.0"
